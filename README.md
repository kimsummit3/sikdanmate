# 식단메이트

식단메이트는 **식단 정보를 많이 주는 앱**이 아니라, **사용자가 실제로 식단을 계속 지키게 만드는 실행 도구**를 목표로 하는 제품입니다.

## 한 줄 정의
친구 같은 음성 식단 메이트가, 사용자의 목표와 상황에 맞춰 **오늘의 식단 결정**을 빠르게 돕고, 실패했을 때도 **다음 한 끼부터 다시 이어주게 만드는 앱**.

## 현재 제품 방향
- 기록형 칼로리 앱이 아님
- 레시피 앱이 아님
- 완제품 식단 커머스가 아님
- **식단 실행 운영체제**에 가까운 제품

## 현재 1차 목표
- 좁은 ICP를 대상으로
- 초저마찰 식단 실행 루프를 검증
- 반복 사용되는 베타 구조 만들기

## 문서 구조
- `docs/` : 전략, ICP, 인터뷰, 메트릭, 로드맵
- `product/` : PRD, MVP 범위, 플로우, 메시지
- `app/` : 실제 구현 또는 프로토타입 코드
- `data/` : 인터뷰 노트, 식단 시나리오
- `ops/` : 실험 로그, 런치 체크리스트
