# mvp-scope-v2.md

## v2 기준 MVP 정의
추천 앱이 아니라, 대화형 운영 루프를 검증하는 MVP를 만든다.

## MVP에 반드시 포함
- 대화/음성 체크인 구조
- 오늘 식단 생성
- 외식/교체/대체 보정
- 가벼운 기록
- 다음 끼니/내일 자동 보정
- 최소 영양 계산

## MVP에 넣지 않을 것
- 처음부터 완전한 월간 플래너
- 매장 단위 최저가 비교
- 정교한 재고/재료 관리
- 가족/멀티유저 구조
- 의료/질환 맞춤 식단

## MVP 이후 빠르게 붙일 것
- 주간 플래너
- 주간 장보기 리스트
- 개인화 메모리
- 예산 모드

## 현재 구현본과 갭
현재 앱은 아래를 이미 갖고 있다.
- 온보딩
- 추천
- 기록
- 요약
- 히스토리
- 로컬 저장

하지만 v2 MVP 기준으로는 아래가 부족하다.
- 대화 우선 UX
- 음성 체크인
- 외식/교체 협상형 흐름
- 자동 보정 중심 홈 구조
- 최소 영양 계산 레이어
