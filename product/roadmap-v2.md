# roadmap-v2.md

## Phase 1 — 매일 쓰게 만들기
- 음성 체크인
- 오늘 식단 1일 생성
- 교체/대체/외식 보정
- 요리 모드 기초
- 최소 영양 계산(칼로리/탄단지/단백질)

## Phase 2 — 돈 내는 이유 만들기
- 주간 식단 자동 생성
- 주간 장보기 리스트
- 개인 취향/회피 메모리
- 예산 모드

## Phase 3 — 월간 운영체제로 확장
- 4주 식단 생성
- 다양성 규칙 반영
- 주간 미세영양소 평균 체크
- 월간 균형 리포트
- 수행률 기반 자동 조정

## Phase 4 — 위치 기반 현실성 강화
- 주소/반경 기반 구매처 추천
- 시장 vs 대형마트 전략 안내
- 온라인 최저가 보조 추천

## Phase 5 — 정밀 가격/재고
- 매장 단위 가격 비교
- 장바구니 자동 담기
- 사용자 구매 데이터 기반 개인화 가격 추천
