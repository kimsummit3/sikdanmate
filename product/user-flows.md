# user-flows.md

## Flow 1 — 온보딩
1. 목표 선택
2. 식단 스타일 선택
3. 기피 음식 / 예산감 선택
4. 오늘 첫 추천 받기

## Flow 2 — 오늘의 식단 제안
1. 현재 상황 질문 (집/밖, 이미 먹음 여부)
2. 오늘 끼니 추천
3. 사용자가 선택
4. 저장 또는 바로 기록

## Flow 3 — 기록
1. 식사 후 사진 또는 버튼
2. 잘 지켰음 / 비슷함 / 벗어남
3. 즉시 다음 끼니 가이드 반영

## Flow 4 — 실패 후 복귀
1. 사용자가 식단 이탈
2. 앱이 죄책감 없이 다음 끼니 기준 재제안
3. 복귀 성공 여부 기록

## Flow 5 — 주간 피드백
1. 기록률 확인
2. 유지율 확인
3. 실패 후 복귀 횟수 확인
4. 다음 주 한 줄 가이드 제공
