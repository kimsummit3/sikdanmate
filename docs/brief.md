# brief.md

## 제품명
식단메이트

## 제품 정의
식단메이트는 사용자가 식단을 **알게 하는 앱**이 아니라, 식단을 **지속하게 만드는 앱**이다.

핵심 가치는 세 가지다.
1. 오늘 무엇을 먹을지 빠르게 결정하게 돕는다.
2. 기록 피로를 줄인다.
3. 실패 후에도 다음 끼니부터 다시 이어주게 만든다.

## 왜 지금인가
- 한국 1인가구 증가
- 건강/체중관리 수요 지속 증가
- 식품/음식 온라인 소비 증가
- 정보는 많지만 식단 실행 시스템은 부족함

## 현재 전략 판단
식단메이트는 초기부터 거대한 영양 플랫폼처럼 넓히면 안 된다.
초기엔 **직장인 식단 실패 반복층**에게 집중해,
반복 사용되는 행동 루프를 증명해야 한다.

## 성공 조건
- 첫 주 반복 사용
- 낮은 입력 피로
- “다음 한 끼 복귀” 경험
- 식단 지속률 개선
